# Shopee Sales Analytics Dashboard

This project transforms retail data into clear business intelligence using **Excel**.  
It tracks KPIs such as Orders, Quantity, Revenue, Average Rating, and Delivery Days, while providing customer insights on buying preferences, product popularity, and regional demand.

# Features
- 📈 Trend analysis (last 13 weeks)
- 🛒 Customer buying behavior (gender & platform)
- 🔥 Product popularity
- 🌍 Regional demand (Indonesia map)
- 🚚 Delivery performance
- 😀 Customer satisfaction metrics
